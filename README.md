## üì¶ Setup & Installation
```bash

```

## üöÄ Deployment to GitHub Pages
```bash
npm install gh-pages --save-dev
```
In `package.json`:
```json
"homepage": "https://krnchittora2.github.io/portfolio",
"scripts": {
  "predeploy": "npm run build",
  "deploy": "gh-pages -d dist"
}
```
Then:
```bash
npm run deploy
```
